# Logo Generator ร้านค้า

เว็บ React + Vite สำหรับสร้างตัวอย่างโลโก้ร้านแบบง่าย ๆ และพร้อมเอาขึ้น GitHub / Vercel

## วิธีใช้งานในเครื่อง

```bash
npm install
npm run dev
```

## Build สำหรับ deploy

```bash
npm run build
```

## อัปขึ้น GitHub

```bash
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/USERNAME/REPO.git
git push -u origin main
```

## Deploy บน Vercel
- เข้า Vercel
- กด New Project
- เลือก repo นี้
- Deploy ได้เลย

เวอร์ชันนี้ยังไม่ได้เชื่อม AI จริง เป็นโครงสำหรับเริ่มต้นและทดสอบ UX ก่อน
